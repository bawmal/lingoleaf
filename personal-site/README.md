# Personal Site (Static)

This folder contains a simple buildless personal landing page.

## Run locally

Open `personal-site/index.html` in your browser.

## Customize

Edit:
- `personal-site/index.html` (text + links)
- `personal-site/styles.css` (theme + layout)

## Deploy

You can deploy this folder as a static site (Netlify, GitHub Pages, etc.).
